# Wordle React Clone Solver

This repository contains a a solver for the popular game Wordle

Link: [here](https://gabrielwoo.github.io/wordle-react-clone-solver/)

NOTE: This application is meant to be used together with Wordle

## How to Use
1. Fill up a word that was previously inputted
2. Click on the letters that were highlighted in the main game 
  (1 click -> yellow, 2 click -> green, 3 clicks -> black)
3. Click Add Guess
4. Repeat Steps 1-3 till satisfied
5. Under 'Possible Words', you may view the possible words remaining based off the words that were given
